Hack

InternHacks is a hackathon focused solely on personal website development. No teams, no worries. Just come and work on that portfolio you haven't updated in a while.

Learn

New to web development? Want to learn a new web framework, how to host a site, or about UI/UX Design?

Throughout the day, InternHacks will be holding workshops on various web dev topics, so you can brush up on the skills needed to make a great personal website.

Network

InternHacks is open to tech interns from throughout the Bay Area, and is a great way to get to know interns from other companies and universities.
